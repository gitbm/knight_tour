Knight Tour Rules

  • Move around the board like a knight chess piece.
  • You can't land on a position you have already been.
  • Try to see the most number of moves you can make.
  • All 64 is possible (or so I have read).
  • The coloured squares should help you see good and bad possibilities.
    • Light Blue - current position
    • Green - multiple moves possible from this square
    • Light Green - only one possible move from this square
    • Red - No possible moves from this position - Game Over
    • Pink - Only one more move will be possible from this square
  • Undo and Reset can help when you get stuck.
  • Have fun!
}
